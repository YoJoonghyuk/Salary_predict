# Salary Predictor 

Проект по предсказанию зарплат на основе данных резюме с использованием машинного обучения.

##  Описание проекта

Цель этого проекта — продемонстрировать возможность построения базовой модели машинного обучения для прогнозирования 
зарплатных ожиданий. В качестве модели используется LinearRegression. 

##   Основные этапы пайплайна:
1. Загрузка подготовленных данных: Использование .npy файлов (x_data.npy, y_data.npy), полученных на предыдущем 
этапе парсинга.
2. Обработка выбросов: Удаление аномальных значений в целевой переменной (зарплате) для улучшения качества модели.
3. Feature Engineering:
  •  Векторизация текста: Текстовые признаки (должность + город) обрабатываются с помощью TF-IDF Vectorizer.
  •  Масштабирование числовых признаков: Числовые признаки (возраст, опыт, зарплата) масштабируются StandardScaler.
4. Обучение модели: Обучение модели LinearRegression.
5. Оценка и сохранение: Оценка качества модели с помощью метрики R2-score и сохранение обученной модели в формате .pkl.
6. Загрузка обученной модели и предсказание зарплат на новых данных.

##  Установка Зависимостей
Убедитесь, что у вас установлен Python 3.8+.
Установите необходимые библиотеки, используя pip:
                   ` pip install pandas numpy scikit-learn joblib`
  ##  Инструкция по запуску
* Шаг 1: **Парсинг данных**

Прежде чем обучать модель, необходимо подготовить данные. Этот скрипт читает исходный hh.csv, извлекает необходимые 
признаки (пол, возраст, опыт, город, должность, зарплата), 
масштабирует числовые признаки, векторизует текстовые и сохраняет результат в data/x_data.npy (признаки) и 
data/y_data.npy (целевые значения).

 `python parse_data.py data/hh.csv`

Результат: в data/ появятся x_data.npy и y_data.npy. В resources/ будут сохранены обученные scaler.pkl и vectorizer.pkl.

* Шаг 2: **Обучение модели**
                            
Этот скрипт загружает x_data.npy и y_data.npy, обучает модель LinearRegression, оценивает её с помощью метрик R^2 и MAE,
а затем сохраняет обученную модель в resources/model.pkl.

   `python train_model.py`

* Шаг 3: **Предсказание зарплат** 
                           
Этот скрипт загружает x_data.npy (или любой другой .npy файл с аналогично подготовленными признаками), загружает 
обученную модель из resources/model.pkl и выдает список предсказанных зарплат (List[float]) в консоль.

   `python app.py data/x_data.npy`

 ##  Вывод
